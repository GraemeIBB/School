## HTML basics
Hyper Text Markup Language

### Syntax:

docs are composed of content and elements

elements encompass: element name, attributes, and content

```html
<a href="http://example.com">Example</a>
```

Empty elements 
```html
<img src="src.png" alt="source image"/>
```

nesting 

```html
<h1>Example <strong>text</strong></h1>
```

hierarchy of elements
```html
<body>
	<div>
		<p>big dog</p>
	</div>
</body>
```
